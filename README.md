## Scrum Product Backlog - BlackBattleship (Papergames.io)

Abaixo encontra-se a lista de User Stories (Requisitos Funcionais) identificadas para a plataforma de Batalha Naval Online, com base nas funcionalidades disponíveis no site.

| ID | User Story | Prioridade |
|:---|:---|:---:|
| **US1** | **Como** jogador, **quero** iniciar uma partida de Batalha Naval contra um bot ou um jogador real (via link), **para que** possa jogar de forma rápida e testar a minha estratégia. | Alta |
| **US2** | **Como** jogador, **quero** poder colocar os meus navios no tabuleiro (de forma manual ou aleatória) antes do início da partida, **para que** possa definir a minha defesa. | Alta |
| **US3** | **Como** jogador, **quero** consultar o meu histórico de partidas (`/match-history`), **para que** possa analisar as minhas vitórias, derrotas e o meu progresso ao longo do tempo. | Média |
| **US4** | **Como** jogador sociável, **quero** adicionar outros utilizadores à minha lista de amigos (`/friends`), **para que** os possa desafiar facilmente para jogos futuros. | Média |
| **US5** | **Como** jogador, **quero** utilizar o chat integrado (`/chat`) durante as partidas, **para que** possa comunicar e interagir com o meu adversário. | Média |
| **US6** | **Como** utilizador competitivo, **quero** criar e participar em torneios (`/t/create-tournament`), **para que** possa competir a um nível mais estruturado com vários jogadores. | Baixa |
| **US7** | **Como** jogador, **quero** aceder a uma loja virtual (`/shop/avatars`, `/shop/emojis`), **para que** possa gastar moedas virtuais em customizações para o meu perfil (avatares e emojis). | Baixa |
| **US8** | **Como** utilizador frequente, **quero** poder aderir a um plano de subscrição (`/pricing/plus`, `/pricing/gruppo`), **para que** tenha acesso a funcionalidades premium e sem interrupções. | Baixa |
